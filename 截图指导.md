3. 点击"📥 加载题目"
4. 在代码编辑器中输入示例代码：

### Q2: 截图后发现质量不好怎么办？
**A:** 重新截图。确保：
- 窗口最大化
- 字体清晰
- 颜色正常

### Q3: AI助手没有反应怎么办？
**A:**
- 检查网络连接
- 查看ai_assistant_widget.py第122行API密钥是否有效
- 等待时间可能较长（2-3秒）

### Q4: 数据库初始化弹窗只出现一次怎么截图？
**A:** 删除数据库文件后重新运行：
```bash
del database\python_learning.db
python main.py
```
