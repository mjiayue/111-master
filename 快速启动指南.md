## 💡 常见问题

### Q1: 提示"python不是内部或外部命令"

**原因：** Python环境变量未配置

**解决方法：**
1. 按 `Win + R`，输入 `sysdm.cpl`
2. 点击"高级"→"环境变量"
3. 在"用户变量"找到`Path`，点击"编辑"
4. 点击"新建"，添加Anaconda路径（例如）：
   - `D:\app\anaconda3`
   - `D:\app\anaconda3\Scripts`
   - `D:\app\anaconda3\Library\bin`
5. 点击"确定"保存
6. **重启电脑**（或关闭所有CMD窗口后重新打开）

---

### Q2: 提示"No module named 'PyQt5'"

**原因：** 缺少PyQt5包

**解决：**
```bash
pip install PyQt5
```

或安装所有依赖：
```bash
pip install -r requirements.txt
```

---

### Q3: 能否把项目发给别人使用？

**完全可以！** 步骤：

1. **复制整个项目文件夹**
2. 对方收到后：
   - 双击 `启动系统.bat`（如已配置Python环境变量）
   - 或按"方法二"手动启动
3. 首次运行会自动初始化数据库
4. 使用账号 `1` 密码 `1` 登录

**无需任何额外配置！**

---

### Q4: 如何重置所有数据？

删除数据库文件后重新初始化：

**Windows：**
```bash
del database\python_learning.db
python init_data.py
```

**macOS/Linux：**
```bash
rm database/python_learning.db
python3 init_data.py
```
